Debian安装MySQL

发表于2025-05-05|更新于2025-05-07
|总字数:133|阅读时长:1分钟|浏览量:

✨使用MySQL APT 仓库安装

这里默认你有linux系统使用经验！

下载APT 仓库

APT 仓库下载地址

右键红框链接并复制链接地址

使用命令：

wget https://dev.mysql.com/get/mysql-apt-config_0.8.34-1_all.deb

进行下载！

安装1.中下载的发布包

sudo apt install ./mysql-apt-config_0.8.34-1_all.deb

更新apt

sudo apt update

安装

sudo apt install mysql-server

安装过程中会让你设置密码

👍验证安装：

出现mysql>表示成功安装！
